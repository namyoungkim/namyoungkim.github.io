# 아키텍처 결정 기록 (ADR)

> 프로젝트의 주요 아키텍처 결정과 그 배경을 기록합니다.
> 새 결정이 필요할 때 기존 결정과의 일관성을 유지하기 위해 참조합니다.

## 작성 형식

```
## ADR-NNN: 제목
- **날짜**: YYYY-MM-DD
- **상태**: 제안됨 / 승인됨 / 대체됨 (ADR-XXX에 의해)
- **맥락**: 어떤 상황에서 이 결정이 필요했는가
- **결정**: 무엇을 선택했는가
- **이유**: 왜 이 선택을 했는가 (대안 대비)
- **결과**: 이 결정으로 인해 어떤 영향이 있는가
```
